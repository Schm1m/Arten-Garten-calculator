# Arten-Garten-calculator
a mobile application designed to help you count your games score by simply taking a few pictures

### Basic idea
- play the game normaly without any changes.
- scan the shared achievements board (to take additional points into the calculation)
- scan each players board an select the players color
- input any additional points gained
- get a summary of points for all players
- (optional) advanced statistics displayable after the game
